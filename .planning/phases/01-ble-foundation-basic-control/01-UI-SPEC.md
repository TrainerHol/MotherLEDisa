# Phase 1 — UI Design Contract

> Visual and interaction contract for BLE Foundation & Basic Control. Generated by gsd-ui-researcher.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | Jetpack Compose (native Android) |
| Preset | Material 3 via Compose BOM 2025.12.00 |
| Component library | Material 3 (androidx.compose.material3) |
| Icon library | Material Icons (built-in with Compose) |
| Font | System default (Roboto on most Android devices) |

---

## Spacing Scale

Declared values (8-point grid, all multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | Compact element spacing, list item padding |
| md | 16px | Default element spacing, section padding |
| lg | 24px | Section breaks, card internal padding |
| xl | 32px | Layout gaps, major component separation |
| 2xl | 48px | Touch targets (minimum), major section breaks |
| 3xl | 64px | Power button size, page-level spacing |

Exceptions:
- Touch targets: Minimum 48px for all tappable elements (Material Design guideline)
- Power button: 64px+ diameter for prominence (D-09)
- Color swatches: 48px diameter for touch accessibility (D-11)

---

## Typography

Material 3 type scale (system defaults):

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 16px | 400 (Regular) | 1.5 (24px) |
| Label | 14px | 500 (Medium) | 1.43 (20px) |
| Title | 20px | 500 (Medium) | 1.2 (24px) |
| Display | 28px | 400 (Regular) | 1.14 (32px) |

Usage mapping:
- Body: Device names, effect descriptions, empty state copy
- Label: Badges ("Connected", "Last connected"), slider percentages, section labels
- Title: Screen titles, section headers ("Color", "Brightness", "Effects")
- Display: Not used in Phase 1 (reserved for Phase 2+ timeline editor)

---

## Color

Dark theme with dual accent colors:

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #121212 | Background surfaces |
| Secondary (30%) | #1E1E1E | Cards, device list items, effects list |
| Primary Accent (10%) | #7C4DFF (Deep Purple) | Connected status, active effects, selected states |
| Secondary Accent | #FFEB3B (Yellow) | Power ON state, scan active indicator |
| Destructive | #CF6679 (Material Dark error) | Disconnect action |

Accent reserved for (explicit list):
- #7C4DFF (Purple): Connected badge, active effect highlight, brightness slider track, selected device outline
- #FFEB3B (Yellow): Power ON button glow, scanning animation pulse, color swatch selection ring (when white selected)
- Destructive (#CF6679): Disconnect button only

On-surface colors:
- Primary text: #FFFFFF (87% opacity)
- Secondary text: #B3B3B3 (60% opacity) for labels, "Last connected" timestamps
- Disabled text: #666666 (38% opacity)

## Interaction Patterns

### Device List Screen

| Interaction | Response |
|-------------|----------|
| Screen visible | Start continuous BLE scan (D-01) |
| Screen hidden | Stop BLE scan |
| Tap device | Show loading indicator, attempt connection (D-03) |
| Long-press device | Show rename dialog (D-16) |
| Tap empty state | Restart scan |

### Control Screen

| Interaction | Response |
|-------------|----------|
| Tap power button | Toggle power, button color changes (yellow ON, gray OFF) |
| Drag color wheel | Continuous color updates to device, preview updates immediately (D-10) |
| Tap color swatch | Set color immediately |
| Drag brightness slider | Debounced to 30fps, preview updates immediately (D-10) |
| Tap effect item | Activate effect, show speed slider (D-20) |
| Drag speed slider | Adjust effect speed immediately |
| Change device picker | Switch control focus, both devices remain connected (D-15) |
| Select "All devices" | Commands sent to all connected devices (D-12) |

## Visual States

### Device List Item

| State | Visual |
|-------|--------|
| Available (not connected) | White text, no accent color |
| Available (known) | White text + "Last connected" secondary text |
| Connecting | Loading spinner replaces trailing content |
| Connected | Purple (#7C4DFF) "Connected" badge |
| Error (connection failed) | Red error icon, error text below name |

### Power Button

| State | Visual |
|-------|--------|
| OFF | Gray background (#666666), "OFF" text |
| ON | Yellow glow (#FFEB3B), "ON" text, subtle pulse animation |
| Transitioning | Loading spinner overlay |

### Effect Item

| State | Visual |
|-------|--------|
| Inactive | Default ListItem appearance |
| Active | Purple (#7C4DFF) background tint, checkmark icon |

### Color Swatch

| State | Visual |
|-------|--------|
| Unselected | 1px gray border |
| Selected | 3px purple (#7C4DFF) border |

## Tower Preview Visualization

Custom Canvas composable rendering vertical tower (D-07):

| Element | Specification |
|---------|---------------|
| Segment count | Match physical tower (typically 5-7 segments) |
| Segment shape | Rounded rectangles, 8px corner radius |
| Segment gap | 4px between segments |
| Tower width | 60% of available width |
| Segment colors | Live from device state or user selection |
| Tower base | Dark gray circle below bottom segment |
| Animation | Smooth color transitions (300ms) |

## Signal Strength Bars

RSSI to bars mapping (Claude's discretion per CONTEXT.md):

| RSSI Range | Bars Filled |
|------------|-------------|
| >= -50 dBm | 4/4 (excellent) |
| -50 to -65 dBm | 3/4 (good) |
| -65 to -80 dBm | 2/4 (fair) |
| < -80 dBm | 1/4 (weak) |

Visual: 4 vertical bars increasing in height, filled bars use white, unfilled use 38% opacity.

## Accessibility

| Requirement | Implementation |
|-------------|----------------|
| Touch targets | Minimum 48dp for all interactive elements |
| Color contrast | WCAG AA minimum (4.5:1 for text, 3:1 for large text) |
| Screen reader | contentDescription for all icons and non-text elements |
| Power button | contentDescription: "Power {state}" where state is "on" or "off" |
| Color swatches | contentDescription: "{color name}" |
| Signal bars | contentDescription: "Signal strength {level}" where level is "excellent/good/fair/weak" |
